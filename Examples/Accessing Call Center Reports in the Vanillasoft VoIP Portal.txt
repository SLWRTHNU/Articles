<p>
  <strong>Call Center Reports</strong> in the Vanillasoft VoIP portal provide detailed
  insights into queue performance, agent activity, and call metrics. These reports
  help you analyze contact center operations, measure individual performance, track
  abandoned calls, and identify trends in call volume and wait times. By reviewing
  call center reports regularly, you can optimize staffing, improve service levels,
  and ensure your team meets performance targets.
</p>
<p>
  Reports are accessed through the Vanillasoft VoIP portal and can be filtered
  by date range, agent, or queue. Once generated, reports can be exported or printed
  for further analysis or record-keeping.
</p>

<div class="accordion">
  <p>
    <strong>Available Call Center Reports</strong>
  </p>
</div>
<div class="panel">
  <p>
    The following call center reports are available in the Vanillasoft VoIP portal.
    These reports help you analyze contact center performance and individual
    agent activity:
  </p>
  <ul>
    <li data-list-item-id="ead8f02fbdd0b99689fb14ecda403817a">
      <p>
        <strong>Abandoned Calls</strong>: This report shows all calls that
        were abandoned in the queue, including the time the call came in,
        time it abandoned, which queue, client name, and phone number. You
        can select a date range to run this report.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="eade99e79e826e2e13eebe9ead3cb6921">
      <p>
        <strong>Agents-Performance</strong>: This report provides detailed
        insights into individual performance, including inbound/outbound
        calls, abandoned calls, average talk time, and longest talk time.
        You can export or print the report once it has been generated.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="e49ea9e5208c145d984601d20ba524695">
      <p>
        <strong>Agents-Status Log</strong>: This report shows individual
        instances when an agent selected a status. You can use the date range
        filter to view the agent's name, start and stop time of the status,
        status selected, and duration. This is a helpful tool to ensure productivity
        and scheduling compliance.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="e0dcd6166209a11f94e83d7f94e95309e">
      <p>
        <strong>Agents-Status Summary</strong>: Similar to the Agents-Status
        Log, this report provides a quick view of the total duration of time
        spent in each status.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="e41e4e528e4194ec5c415aa3e92d95161">
      <p>
        <strong>Callback Failed</strong>: This report shows which calls did
        not trigger a callback based on your call center configuration. For
        this report to work, you must ensure that the callback feature is
        enabled.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="e23f82826723aa9e56b23df277c8122b8">
      <p>
        <strong>Calls to Voicemail</strong>: This report shows which calls
        were not answered in the queue based on your configured max wait
        times. If your call center has a voicemail fallback, this is where
        you will see a list of callers who left a voicemail to be followed
        up on.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="e78a52a04df838a899a0d45fa738201be">
      <p>
        <strong>Limit Waiting Time Exceeded</strong>: This report shows which
        calls waited in the queue the longest beyond the set wait times.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="e26cc9679060c0f7f9229b9b55c13e09d">
      <p>
        <strong>On Hold Less 30 Seconds</strong>: This report shows which
        calls were answered under 30 seconds and which agent answered the
        call.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="e81c1b48afc3a22161212f7d5949cb58d">
      <p>
        <strong>On Hold More 30 Seconds</strong>: This report shows which
        calls were answered above the 30 second threshold and which agent
        answered the call.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="ea29b2324f67f7d9132e96701beab5031">
      <p>
        <strong>Queue-Performance</strong>: This report provides an overview
        of all your queues with total calls, average wait time, abandoned
        calls, and average handle time (AHT). It also calculates what percentages
        of calls were answered, abandoned, and went to voicemail.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="e7c05efe3ee74aea3b423e04509737dfe">
      <p>
        <strong>Unanswered Calls</strong>: This report shows which calls
        were not answered. This is based on a call being offered to an agent
        and queue that went unanswered.&nbsp;
      </p>
      <hr style="height: 1px; visibility: hidden;">
    </li>
    <li data-list-item-id="e8333ba8314129c6406e07142c787b41f">
      <strong>Usage</strong>: You can use this report to drill down by hour
      your call queue traffic.
    </li>
  </ul>
