Ring groups (grouped extensions) allow multiple phone extensions to ring simultaneously or in sequence when a call comes in. This ensures calls are answered quickly even when individual users are unavailable.

Setting up ring groups improves call distribution and reduces missed calls by routing incoming calls to a team of available agents rather than a single line.

Create a Ring Group

  1. Log in to the Vanillasoft VoIP portal.

    • Note: This portal may display Cloudli branding because it is hosted by Cloudli. All configuration and support are handled by Vanillasoft. For assistance, contact Vanillasoft Support.

  2. In the left navigation menu, click Telephony, then select Ring Groups.

  3. Click Add to create a new ring group.

  4. Enter a Name for the ring group and configure the ring group settings as needed.

  5. Use the Members section to add the extensions that should ring when a call comes in for this group.

  6. Click Save to create the ring group.

Once your ring group is set up, you can assign it to a phone number or menu component so incoming calls are routed to the group automatically. For help with related call routing configurations, see Creating a Menu Component in the Vanillasoft VoIP Portal. If you need additional assistance, contact Vanillasoft Support.
