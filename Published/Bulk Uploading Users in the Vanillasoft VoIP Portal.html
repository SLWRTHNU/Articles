Bulk uploading allows you to add multiple users to the Vanillasoft VoIP portal at once using a CSV template, saving significant time compared to creating users one by one. This is especially useful during initial setup or when onboarding a large group of users.

Download the Import Template

  1. Log in to the Vanillasoft VoIP portal.

    • Note: This portal may display Cloudli branding because it is hosted by Cloudli. All configuration and support are handled by Vanillasoft. For assistance, contact Vanillasoft Support.

  2. In the left navigation menu, click Client-Sales, then click Import Persons.

  3. Click Import Template to expand that section of the menu.

  4. Download the correct template for your use case. For voice and VoIP users, select and download Import Person (Voice).CSV.

    • Note: Using the wrong template may cause your import to fail. Confirm you are downloading Import Person (Voice).CSV for VoIP user uploads.

Upload Users in Bulk

  1. Log in to the Vanillasoft VoIP portal.

    • Note: This portal may display Cloudli branding because it is hosted by Cloudli. All configuration and support are handled by Vanillasoft. For assistance, contact Vanillasoft Support.

  2. In the left navigation menu, click Client-Sales, then click Import Persons.

  3. Click Import Persons to expand that section of the menu.

  4. Optional: Check the practice session checkbox to test your upload without committing any changes. The system will report whether the upload would have been successful, allowing you to identify and fix any errors before importing for real.

  5. Choose your completed CSV file, then click Import to begin the upload. Double-check that you are using the correct template (Import Person (Voice).CSV for VoIP users) before proceeding.

Manage Previous Uploads and Export User Data

  1. Log in to the Vanillasoft VoIP portal.

    • Note: This portal may display Cloudli branding because it is hosted by Cloudli. All configuration and support are handled by Vanillasoft. For assistance, contact Vanillasoft Support.

  2. In the left navigation menu, click Client-Sales, then click Import Persons.

  3. Click Previous Imports to expand that section of the menu. A list of your previous uploads will be displayed.

  4. To delete a previous upload, select it from the list and click Delete, then confirm the deletion when prompted.

  5. To export your current user configuration, click the Export Persons tab, then click Download to save the data to your computer.

Once your users have been imported, you will likely need to configure their access levels. See Assigning Rights and Permissions to Users in the Vanillasoft VoIP Portal for instructions on setting individual or bulk permissions. If you need additional help, contact Vanillasoft Support.
