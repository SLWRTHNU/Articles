This article explains how to set up a SCID number in the Vanillasoft VoIP portal. A SCID (Subscriber Channel Identification) number is a unique identifier that can be associated with an extension, allowing the system to track and manage outbound call routing for individual users. Setting up SCID numbers helps ensure accurate call attribution and supports reporting needs across your organization.

SCID numbers are especially useful for organizations that need to track which agent or extension placed a call, making them valuable for reporting and auditing purposes. Once a SCID is assigned, the extension will automatically use it for outbound calling.

Assign a SCID Number to an Extension

  1. Log in to the Vanillasoft VoIP portal.

    • Note: This portal may display Cloudli branding because it is hosted by Cloudli. All configuration and support are handled by Vanillasoft. For assistance, contact Vanillasoft Support.

  2. In the left navigation menu, click Telephony, then select Extensions.

  3. Locate and click the extension you want to assign a SCID number to.

  4. Click Edit to open the extension settings.

  5. Locate the SCID field and enter the SCID number you want to assign to this extension.

  6. Click Save to apply the SCID assignment.

  7. Confirm the SCID number appears correctly in the extension details view.

Once the SCID number is assigned, outbound calls from this extension will be associated with that identifier in your call logs and reports. To learn more about managing extensions and their settings, see Setting Up a New User in the Vanillasoft VoIP Portal. For further assistance, contact Vanillasoft Support.
