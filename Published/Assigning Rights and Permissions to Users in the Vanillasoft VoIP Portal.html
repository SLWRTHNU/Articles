Managing user rights and permissions ensures team members have appropriate access to complete their roles without exposing sensitive admin controls. Permissions can be assigned individually or in bulk using the batch assignment tool.

Assign Permissions to an Individual User

  1. Log in to the Vanillasoft VoIP portal.

    • Note: This portal may display Cloudli branding because it is hosted by Cloudli. All configuration and support are handled by Vanillasoft. For assistance, contact Vanillasoft Support.

  2. In the left navigation menu, click Client-Sales, then click Users.

  3. All users in your account are listed. Click the name of the user whose permissions you want to update.

  4. Click the Rights Management tab.

  5. Click Edit.

  6. Click Add.

  7. Select the account from the dropdown. A user may have access to more than one account, so ensure you select the correct one.

  8. Select the rights and permissions you want to assign to this user.

  9. Click Submit to apply the selected role or permission. To assign multiple privileges, click Add again and repeat the selection and submit process until all required permissions have been added.

  10. When all permissions have been added, scroll to the bottom of the page and click Save.

    • Note: The user must log out and log back in to the portal for the new permissions to take effect.

Batch Assign Permissions to Multiple Users

  1. Log in to the Vanillasoft VoIP portal.

    • Note: This portal may display Cloudli branding because it is hosted by Cloudli. All configuration and support are handled by Vanillasoft. For assistance, contact Vanillasoft Support.

  2. In the navigation menu, click Roles/Rights Batch Assignment. This tool uses a roles wizard to assign roles to groups of users at once.

  3. Select the role or permission you want to assign.

  4. Click Next.

  5. Select the users you want to assign to this role.

  6. Click Next.

  7. Review the summary of changes to confirm the correct role and users are selected.

  8. Click Submit to apply the batch assignment.

After setting up user permissions, you may also want to add users in bulk. See Bulk Uploading Users in the Vanillasoft VoIP Portal for instructions on importing multiple users at once using a CSV template. If you need additional help, contact Vanillasoft Support.
