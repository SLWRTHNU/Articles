This article explains how to generate a call statistics report in the Vanillasoft VoIP portal. The call statistics report provides totals for inbound and outbound calls, call duration, and averages over a selected date range, giving you a clear overview of your organization's calling activity.

Generate a Call Statistics Report

  1. Log in to the Vanillasoft VoIP portal.

    • Note: This portal may display Cloudli branding because it is hosted by Cloudli. All configuration and support are handled by Vanillasoft. For assistance, contact Vanillasoft Support.

  2. In the left navigation menu, click Reports, then select Call Log, then click Call Statistics.

  3. Select the date you want to run the report for using the date selector.

  4. Click Generate to run the report. The results will appear at the bottom of the screen, showing inbound and outbound call totals, duration, and averages.

    • Inbound calls – total calls received during the selected period
    • Outbound calls – total calls placed during the selected period
    • Duration – total and average call length for the period
    • Averages – mean values calculated across all calls in the selected range

  5. To print the report, use your browser's print function or any print option available within the report view.

For a more detailed breakdown of call activity organized by extension, see Viewing Summary Reports by Extension in the Vanillasoft VoIP Portal. If you need assistance interpreting or generating call reports, contact Vanillasoft Support.
