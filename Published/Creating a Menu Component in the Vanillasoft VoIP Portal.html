Menu components are modular building blocks for your call routing system in the Vanillasoft VoIP portal. They allow you to define how incoming calls are handled, including directing callers to ring groups, extensions, voicemail, or other destinations.

Using menu components gives you fine-grained control over your call flow, making it easier to update routing rules without reconfiguring your entire phone system.

Create a Menu Component

  1. Log in to the Vanillasoft VoIP portal.

    • Note: This portal may display Cloudli branding because it is hosted by Cloudli. All configuration and support are handled by Vanillasoft. For assistance, contact Vanillasoft Support.

  2. In the left navigation menu, click Telephony, then select Menu Components.

  3. Click Add to create a new menu component.

  4. Enter a Name for the menu component and configure routing options, including which destination the call should be forwarded to (such as a ring group, extension, or voicemail).

  5. If you want callers to be able to select options using their keypad, configure the Key Options section to assign destinations to each key press.

  6. (Optional) Upload or record an audio greeting that will play when callers reach this menu component.

  7. Click Save to create the menu component.

Once created, the menu component can be assigned to an inbound phone number so that incoming calls follow the routing rules you've defined. To learn more about setting up ring groups that can be used as menu destinations, see Creating a Ring Group in the Vanillasoft VoIP Portal. For further assistance, contact Vanillasoft Support.
